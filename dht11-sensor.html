01 — Overview
What does this module do?

The DHT11 module senses the surrounding air and outputs both temperature (in °C) and relative humidity (%) as a digital signal — meaning no complex analog reading is needed. Just connect three wires, install a library, and you're reading live environmental data within minutes.

03 — Wiring
How to connect to Arduino
1
RED wire VCC → 5V
Connect the VCC pin on the DHT11 to the 5V pin on your Arduino.
2
BLACK wire GND → GND
Connect the GND pin on the DHT11 to any GND pin on your Arduino.
3
GREEN wire DATA → Pin 2
Connect the DATA pin to digital pin 2 on your Arduino. You can use any digital pin — just update the pin number in the code to match.
4
Skip pin 3 (N/C)
The third pin is "Not Connected" — leave it unconnected. It has no function.
04 — Code
Arduino sketch
DHT sensor library
by Adafruit — free, open source

Install the library before uploading the code:

  1. Open Arduino IDE → go to Sketch → Include Library → Manage Libraries
  2. Search for DHT sensor library
  3. Click Install on the one by Adafruit
  4. Also install Adafruit Unified Sensor if prompted
dht11_basic.ino 
// DHT11 Temperature & Humidity Sensor
// Module World — moduleworld.github.io

#include "DHT.h"

#define DHTPIN  2       // DATA pin connected to pin 2
#define DHTTYPE DHT11   // Sensor type: DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
  Serial.println("DHT11 ready!");
}

void loop() {
  delay(2000);  // Wait 2 seconds between readings

  float humidity    = dht.readHumidity();
  float temperature = dht.readTemperature(); // Celsius

  // Check if reading failed
  if (isnan(humidity) || isnan(temperature)) {
    Serial.println("Error: Could not read from DHT11");
    return;
  }

  Serial.print("Temperature: ");
  Serial.print(temperature);
  Serial.println(" °C");

  Serial.print("Humidity:    ");
  Serial.print(humidity);
  Serial.println(" %");

  Serial.println("---");
}
05 — Troubleshooting
Common issues
!
Serial monitor shows "Error: Could not read from DHT11"
▶
This usually means a wiring problem. Double-check that VCC is connected to 5V (not 3.3V), GND is connected to GND, and DATA is on the exact pin number you defined in the code (default: pin 2). Try a different wire or breadboard column — loose connections are the most common cause.
!
Temperature reads 0°C or humidity reads 0%
▶
A reading of exactly 0 usually means the sensor is responding but returning bad data. Make sure you have the DHT library installed correctly (Adafruit DHT sensor library). Also ensure the delay between readings is at least 2000ms — reading too fast confuses the sensor.
!
Nothing appears in the Serial Monitor
▶
Make sure the baud rate in the Serial Monitor (bottom-right dropdown) matches the one in the code — it should be set to 9600. Also confirm you selected the correct COM port under Tools → Port in Arduino IDE.
!
Values seem inaccurate or jumping around
▶
The DHT11 has a ±2°C and ±5% RH tolerance — small variations are normal. If values jump wildly, check that the DATA wire isn't next to a high-power wire causing interference, and consider adding a 10kΩ pull-up resistor between VCC and DATA if you're using the bare sensor (not the module board).
